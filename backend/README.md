# 🩺 AI Health Scanner - ML Powered

A full-stack AI-powered health prediction system built with FastAPI and Flutter.

## 🚀 Live Backend
https://ai-health-scanner.onrender.com/docs

## 🔐 Features
- User Registration & Login (JWT Authentication)
- Secure Protected Endpoints
- Heart Disease Risk Prediction
- Diabetes Risk Prediction
- ML Models integrated using joblib
- SQLite Database
- Deployed on Render

## 🛠 Tech Stack
Backend:
- FastAPI
- SQLAlchemy
- JWT Authentication
- SQLite
- Scikit-learn
- NumPy

Frontend:
- Flutter
- HTTP
- Shared Preferences
- fl_chart

## 🔒 Security
- Password hashing using bcrypt
- JWT token authentication
- Protected API endpoints

## 📦 Deployment
Backend deployed on Render.
